Biblioteca Virtual em Python

Um sistema de gerenciamento de livros feito em Python utilizando persistência em JSON.

 Funcionalidades:
 Cadastro de livros
 Listagem de livros
 Busca de livros
 Edição de livros
 Remoção com confirmação
 IDs únicos automáticos
 Persistência de dados em JSON
 Ordenação automática por ID
 Tratamento de erros básicos
 Interface interativa via terminal
 Tecnologias Utilizadas
        Python 3
        JSON
        Manipulação de arquivos
        Estruturas de dados
        Funções e modularização
